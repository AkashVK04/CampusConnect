# CampusConnect

Smart Campus Management & Placement Portal built using Spring Boot, MySQL, JWT Authentication, and REST APIs.

## Features

- User Registration & Login
- JWT Authentication
- Student Profile Management
- Company Management
- Placement Portal Backend
- MySQL Database Integration

## Tech Stack

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Maven
- Git & GitHub

## APIs

### Authentication

- POST /api/auth/register
- POST /api/auth/login

### Student Module

- POST /api/student/profile
- GET /api/student/profile/{id}
- GET /api/student/profiles

### Company Module

- POST /api/company
- GET /api/companies

## Database Tables

- users
- student_profiles
- companies
